# ことばのキャッチボール

会話でスコアが貯まる2人用リアルタイムブラウザゲーム。

## ローカルで起動する手順

```bash
cd kotoba-catchball
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## スマホで2人が遊ぶ方法

同じWi-Fiネットワーク上にいる場合：
1. PCのIPアドレスを確認（例: 192.168.1.10）
2. `npm run dev` でサーバー起動
3. 1人目: http://192.168.1.10:3000 を開いて「ルームを作る」
4. ルームコードをメモしてURLや口頭で相手に伝える
5. 2人目: 同URLを開いて「ルームに入る」→ルームコードを入力

## 技術スタック

- Next.js 16 + TypeScript
- Tailwind CSS
- Socket.IO (リアルタイム通信)
- Framer Motion (ボールアニメーション)

## 球種

| 球種 | 条件 | 演出 |
|------|------|------|
| ⚾ ストレート | 短めの返答・軽い質問 | まっすぐ速く飛ぶ |
| 🎾 ロブ | 相手を受けて問いが含まれる | ふわっと高く飛ぶ |
| 🌀 スピン | 自分の経験・エピソードあり | 回転しながら飛ぶ |
| 💫 カーブ | 意外な質問・話題転換 | 曲がって飛ぶ |
| ⭐ ゴールデン | ありがとう・いいね・共感 | 光りながら高く飛ぶ |

## スコア設計

- メッセージ送信: +10
- 相手内容を受けている: +10
- 自分の情報を返している: +10
- 質問が含まれる: +10
- 感謝・肯定・共感: +20
- 30文字以上の詳細さ: +15
- 3往復以上コンボ: +10
- 5往復以上コンボ: +20
- 10往復以上コンボ: +30
